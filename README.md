## Build Setup

```bash
# 克隆项目
git clone https://github.com/xiqi99520/fakeappstore.git
# 进入项目目录
cd fakeappstore
# 安装依赖
npm install
# 启动服务
npm run serve
```

浏览器访问 [http://localhost:8080](http://localhost:8080)

## 发布

```bash
# 构建生产环境
npm run build
```
